PicOptim

A Minimal JPEG Optimization Tool

Download Script View on GitHub

Features

  • Lossless optimization (no quality reduction)
  • Lossy optimization (reduce quality)
  • Metadata stripping
  • Parallel processing
  • Backup file creation

Getting Started

1. Install Dependencies

pip install tqdm psutil argparse

2. Install jpegoptim

  • Windows: Download from here
  • Linux: Run sudo apt-get install jpegoptim
  • macOS: Run brew install jpegoptim

3. Run PicOptim

python picoptim.py /path/to/images --lossy --quality 75 --strip-metadata

System Requirements

  • Python 3.6+
  • jpegoptim installed
  • pip installed
