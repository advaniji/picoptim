Installation & Usage Guide

Follow these steps to get started with picoptim:

  1. Download the Files: Obtain picoptim.py and the appropriate jpegoptim binary for your operating system.
  2. Place the Files: Ensure picoptim.py and the jpegoptim binary are in the same directory as your JPEG images.
  3. Install Dependencies: Run: 
    pip install tqdm
  4. Run picoptim: Execute picoptim.py using Python 3: 
    python picoptim.py
  5. Using the Application: The script will process all JPEG images in the directory, optimizing them using jpegoptim.

Ensure you have Python 3 installed. For additional details or troubleshooting, please consult the project documentation on GitHub.

About & Credits

picoptim is a straightforward Python tool designed to optimize JPEG images using the jpegoptim utility. It simplifies the process of reducing image sizes without compromising quality.
